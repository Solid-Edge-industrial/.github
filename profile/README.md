# Solid Edge — Professional Industrial Design and Product Modeling

## About Solid Edge

Solid Edge provides a complete professional industrial design and product modeling environment for mechanical engineering and manufacturing. Solid Edge combines synchronous technology that merges parametric and direct editing workflows. Solid Edge supports sheet metal design, weldments, surface modeling, and assembly management. Solid Edge integrates finite element analysis, motion simulation, and generative design capabilities. Solid Edge enables users to create production-ready components with intelligent manufacturing attributes. Solid Edge is trusted by industrial designers, mechanical engineers, product developers, and manufacturing teams worldwide.
